# AI-Powered Food Redistribution and Supply Chain Optimization for Hunger Relief

## Project Objective

Build a decision-support system that recommends how surplus food should be redistributed to food-deficit regions while minimizing transportation cost, delivery time, food spoilage, and unmet critical demand.

## Key Objectives

- Define a clean and reproducible project structure.
- Prepare raw, processed, and generated data folders.
- Document the study area, datasets, and lineage before implementation.
- Support future backend, frontend, and optimization development.

## Technology Stack

- React.js
- Tailwind CSS
- Django
- Django REST Framework
- PostgreSQL
- Pandas
- NumPy
- NetworkX
- Google OR-Tools
- Scikit-learn
- OpenWeather API
- OpenRouteService API

## Repository Structure

- `backend/`
- `frontend/`
- `data/raw/`
- `data/processed/`
- `data/generated/`
- `docs/`
- `scripts/`
